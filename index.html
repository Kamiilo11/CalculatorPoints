Club points calculator

Welcome to our club points calculator

Please select one of the following options to start calculating

Casino

  • Please bear in mind that the bingo and poker games are treated like slot games, so you should use that option when calculating with that kind of games.
  • The live games like Monopoly or dream catcher are calculated like a roulette, please use the according option.
  • Also the baccarat games have to be calculated using the option for Blackjack.

Sports

  • All those bets whose odds are below 1,4 does not grant any points.
  • The system bets do not acumulate any points.
  • For the points to be reflected on the balance the bet must have been closed either lost or won.
